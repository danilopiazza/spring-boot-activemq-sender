# Running

## ActiveMQ Docker Container

Run the [ActiveMQ Docker Container][1] with:

    docker-compose up

# Monitoring

## ActiveMQ Console

Connect to the ActiveMQ Console URL:

- http://localhost:8161/

using the following credentials:

- User name: `admin`
- Password: `admin`

[1]: https://hub.docker.com/r/rmohr/activemq
